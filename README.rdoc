== README

Thelp is a commnunity based app to help and support those with Thyroid conditions. It has been build in Ruby and Rails. 

The majority of the site is available to anyone howevr, if you want to add or comment on a question you need to be come a user. There is a login using Devise or Google login. 

There is also an admin section which has CKeditor. The site is completely editable via the admin section. It is here that articles can be uploaded.
